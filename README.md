# My Base NFT

A simple ERC721 NFT collection deployed on Base mainnet.

## Features
- ERC721 standard NFT
- Max supply of 100 NFTs
- Mint price of 0.01 ETH
- Whitelist for free minting
- Owner can pause/unpause minting
- Owner can withdraw ETH
- Event logging for all actions
- Metadata URI support

## Contract
- **Standard:** ERC721
- **Network:** Base Mainnet
- **Max Supply:** 100
- **Mint Price:** 0.01 ETH

## Built With
- Solidity ^0.8.20
- OpenZeppelin Contracts
- Remix IDE
